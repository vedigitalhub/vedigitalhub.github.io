Preço Real

Política de Privacidade

Última atualização: 19 de junho de 2026

Esta Política de Privacidade descreve como o aplicativo Preço Real, desenvolvido por VeDigitalHub, coleta, usa e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com as práticas descritas neste documento.

1. Dados coletados

O Preço Real pode coletar as seguintes informações:

  • Dados de autenticação: e-mail e nome, quando o usuário cria uma conta ou faz login com Google ou Apple.
  • Dados de compras: informações extraídas de notas fiscais eletrônicas (NFC-e) escaneadas pelo usuário, como nome de produtos, preços e data da compra. Esses dados ficam armazenados localmente no dispositivo.
  • Dados de localização: localização aproximada, usada para identificar mercados próximos. Coletada apenas com permissão explícita do usuário.
  • Dados colaborativos anônimos: preços de produtos compartilhados de forma anonimizada com outros usuários, sem qualquer identificação pessoal.

2. Uso das informações

As informações coletadas são usadas para:

  • Registrar e exibir o histórico de compras do usuário;
  • Comparar preços entre diferentes estabelecimentos;
  • Sugerir mercados mais econômicos com base no histórico real;
  • Melhorar os nomes dos produtos via inteligência artificial (DeepSeek API), de forma anônima;
  • Alimentar a base colaborativa de preços, anonimamente.

3. Compartilhamento de dados

O Preço Real não vende dados pessoais a terceiros. Os dados podem ser compartilhados apenas com:

  • Firebase (Google): autenticação e armazenamento de dados em nuvem. Política do Firebase.
  • DeepSeek API: nomes de produtos abreviados (sem qualquer dado pessoal) são enviados para normalização via IA.
  • SEFAZ (Secretaria da Fazenda): consulta pública de notas fiscais a partir do QR Code escaneado pelo usuário.

4. Armazenamento e segurança

Os dados de compras são armazenados primariamente no dispositivo do usuário (banco de dados local SQLite). Dados sincronizados com a nuvem utilizam infraestrutura Firebase com criptografia em trânsito (TLS/HTTPS) e em repouso.

5. Retenção de dados

Os dados são mantidos enquanto o usuário utilizar o aplicativo. O usuário pode excluir sua conta e todos os dados associados a qualquer momento pelas configurações do app.

6. Direitos do usuário

O usuário tem direito a:

  • Acessar e exportar seus dados;
  • Corrigir informações incorretas;
  • Solicitar a exclusão de sua conta e dados;
  • Desativar o compartilhamento colaborativo de preços nas configurações do app.

7. Crianças

O Preço Real não é direcionado a crianças menores de 4 anos e não coleta intencionalmente dados de crianças.

8. Alterações nesta política

Esta política pode ser atualizada periodicamente. Notificaremos os usuários sobre mudanças significativas dentro do próprio aplicativo.
